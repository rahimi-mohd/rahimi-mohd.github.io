Arch Linux Experiences

I have been an avid linux user since late of 2019 due to my obsession with Mr.Robot and my hacker wannabe cringey teenager. Long story short, I failed on my attempt to be a hacker obviously and after many distro later I stick with Linux Mint. The reason is Linux Mint is so easy to setup and my familiarity with Ubuntu make it eaiser to used it.

I also happen to own several old laptop that have been collecting dust since I don't know when and I think it just normal for linux user to use their old laptop to try other distro or distro hop. To be honest, I really like to distro hop because the feels you get when you manage to install it successfully on your bare metal laptop is so satisfying. I have tried some major distro such as Debian, Ubuntu, Linux Mint, Fedora, Manjaro, Endeavour OS, Arch, and some not so famous one like Void (I'm sorry if they are famous), LXLE and many more.

The point is, I have spend countless hour to install and setup my freshly installed linux distro and repeat the process too many times and I will always go back to Linux Mint. But, this is not about my linux journey or how I really really love Linux Mint, this is about my transition from Linux Mint to Arch Linux.

The story goes like this, I have tried Arch Linux multiple times before and my setup is basically the same every time, Arch + i3. I really love tiling windows and i3 happen to be the first I tried and I don't bother to change to other tiling window manager for whatever reasons, i3 just work for me and I really love the workflow. But as I said before, I always go back to Linux Mint because it's really feel like home to me. There's a time when I used Arch for a month or something then I changed back to Linux Mint for no reason, that how I love Linux Mint.

But, I have this habit that I like to start fresh every time I start a new semester. Yes I'm old but I'm still a student. So this time I think to myself let's try to use Arch for this whole semester and the journey start on September 2023.

The installation is quiet easy since I've do it so many times and I have some script for which packages to install during installation and all. To be honest, the installation process is not so hard and everyone can do it if they can do it properly, just remember to install network manager and boot manager and you will be safe.

I remember the reason why I love Debian based distro is because it is easier to find .deb packages. The packages that I have to installed no matter what is Visual Studio Code and Google Chrome for college, and pacman doesn't have both. So that's is the main reason why I used Debian based distro before..... I know how to use AUR properly. Both Visual Studio Code and Google Chrome can be install using AUR and I used yay to manage my packages.

The way I setup my system is by using alacritty as my main terminal emulator and I set it on my first workspace, second workspace is for browsers such as Firefox (For my personal browsing stuff) and Google Chrome (For college things only). Then I have Visual Studio Code to run on workspace 3 and other software on other workspaces.

This ensure my productivity and easier for me to do things when I can switch workspaces like that. I also used tmus and neovim(lazyvim) for coding personal project and editing text. It's not like I'm good at vim, I just love the keybinding especially it movement. I also use vim extension on Visual Studio Code because I used to it.

For file manager, I use pcmanfm because it's really lightweight. Actually, file manager and mount thinggy was also the reason I changed distro back then, I used external hard disk a lot and I don't know how to automatically mount it, I have to sudo mount /dev/blabla /mnt/blabla every time and it can get very tired right?! I know I'm stupid, the only thing that I have to install is gvfs and it help to mount the drive automatically so my problem is solved.

Other software that I use is Okular for PDF viewer, this is very important for me since all my college's resources and book are in PDF so I choose Okular because it's easy to use. VLC for video and music player, actually I didn't listen to music much and I use Spotify whenever I want to listen to music. Thank again for AUR.

It's been almost a month since I use Arch and I hope that I can maintain and use it atleast until January which is end of this semester, then maybe I can change to other distro or back to my beloved Linux Mint. I also planned to keep updating about my journey with Arch Linux or....... my journey with other distro if I change to other distro.

For now Arch is very good and I start to maintain it very well, the most important thing is it have all the software needed for my college. It will be pointless if it didn't support the software needed for my college.
