Programming

Post on Tue, 31 Oct 2023

I learn a lot of technology stuff by wanting to learn programming.

 I started to learn programming back in 2019 when I'm nearly 30 years old. I always think that I'm kinda late in the game and that's one of the thing that I regret in life. I always love technology stuff when I was a kid, my family own a computer when I am 7 - 8 years old but I never learn programming back then. It's not anybody fault, it just I don't find it that time that I really love to code.

 Don't get me wrong, it's not like I'm a pro coder or I've built some life changing applications, it's just I really like to code and learn new programming languages. Languages that I have my interest on right now is Python and Rust. The first programming language I learn back then was C++ and I failed, I didn't make it pass variable section because I'm busy that time with my full time work and life responsibilities (C++ was very hard too :'( ).

The first C++ code I wrote is this simple profit calculator, life changing right ?! 


                    int main() {
                        string whatToSell;
                        float modal;
                        float todaySales;

                        cout << "Tell me what do you want to sell: ";
                        cin >> whatToSell;

                        cout << "This is your " << whatToSell << " business calculator\n";

                        cout << "Enter your modal: ";
                        cin >> modal;

                        cout << "Enter today sales: ";
                        cin >> todaySales;

                        cout << "Your profit for today is " << todaySales - modal << ".";

                        return 0;
                    }

 I stop programming for awhile and then I came back with Python, that's the best time of my life and I really enjoy the learning process, I learn most of programming concept by learning python. I learn about variable, loop, function, oop and many more. This is not an advertisement or promoting Python programming language, but I really recommend people who want to learn programming (and not have interest in web programming) to learn Python. This is because it help to teach you the concept of programming language properly, after that, you can move on to any language that you like. For people who like web development especially frontend, you can start by learning javascript.

 While learning Python, I heard about Linux and I become curious, really curious. I started to dig more about Linux, I have my own computer that time (late of 2019) and I installed Ubuntu out of curiosity, I never looked back at Windows since then. I don't remember what version of Ubuntu I used back then, but I installed the latest version (non lts) and I really love it back then, it run smooth, there's so many things to do and I really love learning about it. I started to learn about command line and the power of it, how I can use my computer with only command line. Linux is becoming my new best friend that time.

 The problem is, I have this new addiction which is Linux, I started to distro hopped a lot and well, I kinda stop learning programming and start to try various flavor of distros available. After many years, I have this realization that it's actually not about the distro, it's more about the desktop environment. I'm most comfortable with Cinnamon and Xfce DE but I use i3 the most. The reason is, I really like it and it improve my productivity.

 Now, I really want to get back into programming. So last year, I register with one of the university in my country as diploma in information technology student (part time). I learn a ton of new things such as computer architecture, operation system, mathematic for it and many more technology stuff, but they only teach the basic of programming using C and Java. I heard about Rust and look into it, it is really interesting and I planned to learn it during semester break, I can't learn it now as I have to finish my assignment and prepare for my final exam (school huh?)

 But, I still learn Python in my free time, I try to learn algorithm with Grokking algorithm book. For me, it is a fantastic book and if you're beginner looking for algorithm book, you should check it out. The book is fun and easy to understand, it teach you how to write in pseudo code and give you the idea how to write the algorithm.

 There's so many things that I want to learn and I'm curious about, the process of learning never stop. What my next plan ? I want to try FreeBSD, it's really interesting and I have this temptation to try it now, but I have to wait until I finish my assignment and my final exam. I don't want to spend so much time in playing around with FreeBSD. I also want to start learning Rust, maybe I will use FreeBSD to learn Rust. And lastly, I want to record the process of learning FreeBSD and Rust and maybe upload it to YouTube, actually I really like live coding and tech stuff like Geo Hortz and Tsoding, I really adore these two and they are very brilliant.

P/S I really hate it when hiring company ghost you, it is so hard to update candidate whether they get the job or not ?!
